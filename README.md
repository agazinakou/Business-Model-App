## Requirement

You must be sure to have the following elements installed on your machine.

- [Bow Framework](https://bowphp.github.io) by [@franck_dakia](https://twitter.com/franck_dakia)
- PHP >= 7
- OpenSSL PHP Extension
- PDO PHP Extension
- Mbstring PHP Extension
- XML PHP Extension
- JSON PHP Extension
